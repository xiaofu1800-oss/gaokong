# 中国高空作业网

中国高空作业网是一个面向高空作业行业的信息发布与机械设备服务平台，主要提供高空作业工程建设企业、高空作业车、脚手架、升降平台、升降机、高空吊篮等相关信息入口。

## 网站介绍

中国高空作业网面向高空作业工程企业、设备厂家、供应商、采购商和相关从业者，提供高空作业行业信息发布、设备信息展示、供求信息、工程服务、技术资料和商贸合作入口。
